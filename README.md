# The _Bluff_ card game

This is a text-based version of the popular _Bluff_ card game in a terminal.

## Usage

To play the game just clone the repository and run the project:
```shell
  cargo run
```

## Known issues
Currently, it is work in progress and not fully functional.  
Main issues:
* The main game loop requires polishing
* The prints throughout the game need to be polished
* Debug strings are present for testing purposes
* Thorough testing is yet to be performed
* The rules are not fully implemented (i.e. You cannot add parameters to a given bet)
